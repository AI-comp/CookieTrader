# Cookie Trader

Welcom to Cookie Trader

## How to prepare develop environment
1. Install node.js (http://nodejs.org/)
2. Install dependecies  
```npm install```
3. Install CoffeeScript  
```npm install -g coffee-script```
4. Run run.bat / run.sh

## Requirements

- node.js v.0.10.19
